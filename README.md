# 🍰 Cake Surgeon

Cake Surgeon is a responsive cake shop website created using HTML, CSS, and JavaScript.  
This website showcases delicious cakes, pastries, customer reviews, and a quick order system for customers in Gorakhpur.

## 🚀 Features
- Beautiful homepage with bakery introduction
- Menu section with cakes and pastries
- Customer reviews
- Contact and location details
- Quick order form
